# Pet Health Decoder

> Educational website translating veterinary terminology into plain English for pet parents

## Tech Stack
- **Framework:** Astro 5.x
- **Styling:** Tailwind 4
- **Search:** Pagefind (client-side)
- **Language:** TypeScript
- **Hosting:** Cloudflare Pages

## Content
- 12 published health articles (Markdown with frontmatter)
- SEO optimized
- Warm UX (orange/pink palette)

## Development
```bash
npm install
npm run dev
```

## Content Structure
Articles in `src/content/articles/` with frontmatter:
```yaml
---
title: "Article Title"
description: "Brief description"
pubDate: 2024-01-15
tags: ["dogs", "health"]
---
```
