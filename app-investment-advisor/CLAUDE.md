# Investment Advisor

> CLI tool for portfolio tracking, DCA planning, and investment analysis

## Tech Stack
- **Runtime:** Node.js + Express
- **CLI:** Commander.js
- **Charts:** Chart.js
- **Data:** Yahoo Finance 2 API

## CLI Commands
```bash
invest              # Main CLI entry point
invest portfolio    # View portfolio
invest analyze      # Run analysis
invest dashboard    # Launch web dashboard
```

## Key Features
- Portfolio analysis
- Stock/ETF tracking
- Rebalancing recommendations
- Price monitoring
- Web dashboard visualization

## Development
```bash
npm install
npm link    # Install CLI globally
invest --help
```
